# AI Productivity OS

AI Productivity OS is an ongoing productivity platform designed to help users manage tasks, track habits, plan schedules, and receive personalized productivity recommendations.

---

## Tech Stack

- Java
- Python
- PostgreSQL
- Git & GitHub
- Linux (Ubuntu/WSL)

---

## Features Implemented

- PostgreSQL Database Schema Design
- Modular Backend Architecture
- Java Model Classes
- Python-based Productivity Recommendation Engine
- Project Documentation

## Recommendation Engine

The recommendation engine analyzes:

- Tasks Completed
- Habit Completion Rate
- Weekly Productivity Score
- Monthly Productivity Score

and generates productivity recommendations.
